Exercise one:

Within the runExercise() function in the JavaScript file, you will see two arrays: languages and descriptions. You should loop through these arrays to display them on the screen so that:

  • Each language from the languages array is displayed within an h3 element
  • The description for each language is displayed below the matching h3 element within a paragraph element
  • When complete, each language should display with the title as an h3 and description as a paragraph directly below
  • You can display the content within the div with the id 'displayExercise'

Suggested steps:

  • Get required references for any HTML elements you may wish to add content to
  • Loop through the languages array with a traditional for Loop
  • For each iteration of the loop, use the 'innerHTML' property of the parent div with '+=' to add the required h3 and paragraph elements to the page with the appropriate data from the arrays nested inside each element
  • If you are unsure of any steps, check the code in the example as it is relatively similar
  • Remember that you can use the 'i' value within a for loop to access the item in that index of a matching array and the languages and descriptions content match each other

Exercise two:

You will see that inside the runExerciseTwo() function, there are a series of arrays: emotions, adjectives, nouns, verbs, adverbs, locations.

For this exercise, you will use the arrays provided to create a random poem generator. When the user clicks the 'Run exercise two' button a random poem should be generated by selecting one word from each array and concatenating/adding it into a poem template you create.

An example poem could be:

  • [emotion] is like a [adjective] [noun] [verb] [adverb] [location].
  • anger is like a bewildered goldfish shivering elegantly behind the refridgerator.

You will need to investigate how to generate random numbers and generate a random number between 0 and the last item of each array to select a random item from each array. Try to write your code so that it automatically adapts to whatever size each array is and won't break if items are added or removed from any array.

For extension you can try to add a few more arrays of different types of words to extend the poem. Try to be creative. Alternatively, you could try to create a random rock song generator or anything else that builds upon the same concept.
